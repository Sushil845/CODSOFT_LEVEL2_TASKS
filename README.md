# CodSoft Web Development Internship

This repository contains my Level 2 Web Development internship projects completed as part of the CodSoft Internship Program.

## Projects

### 1. Job Board
A full-stack job portal where employers can post jobs and candidates can search and apply for jobs.

### 2. Online Quiz Maker
A full-stack web application that allows users to create quizzes, take quizzes, and view results.

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- HTML
- CSS
- JavaScript

The projects are being developed progressively with regular Git commits throughout the internship.
